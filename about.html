Goal

Streaming Services are becoming the new cable package. Combining multiple streaming services can become costly, and more difficult to manage. The main goal of this website is to be a tool to help users determine which streaming service is the best. There are many blogs that express their opinion of the most useful streaming service, but nothing that reflects the data behind each provider. This tool provides diagrams of statistical data based on user selection to provide a meaningful experience personaly.

Data

The data in use has been pulled from streaming service databases, and combined with data from IMDB, and Rotten tomatoes.
